# 🌦️ India Weather Live

A beautiful, real-time weather dashboard tracking over 95 cities across India, from major metropolitan hubs to remote interior states. 

### ✨ Features
* **Real-Time Data:** Accurate, up-to-date weather data synced directly from Open-Meteo's global forecast models.
* **Premium Aesthetics:** "Aurora Borealis" theme featuring a dynamic dark mode, glassmorphic cards, and a custom SVG paper-plane cursor with trailing animations.
* **Deep Insights:** Hourly forecasts (next 24h), 7-day weekly outlooks, UV index, dew point, cloud cover, and visibility tracking.
* **Personalized:** Save your favorite cities locally, sort by temperature/humidity, and toggle between °C and °F.
* **No Server Needed:** 100% client-side architecture using modern JavaScript (ES6+).
